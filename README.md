# Active Directory Home Lab

## Project Overview

This project documents my hands-on Active Directory home lab built to develop practical skills in Windows Server administration, networking, and enterprise IT infrastructure.

## Lab Environment

- Windows Server 2022 (Domain Controller)
- Windows 10 Pro (Domain Client)
- Kali Linux (Testing Workstation)
- VirtualBox

## Skills Demonstrated

- Installed Windows Server 2022
- Configured Active Directory Domain Services (AD DS)
- Configured DNS
- Created the lab.local domain
- Created Organizational Units (OUs)
- Created domain administrator accounts
- Joined a Windows 10 client to the domain
- Verified domain authentication
- Performed network and DNS troubleshooting
- Tested connectivity from Kali Linux

## Technologies

- Windows Server 2022
- Active Directory
- DNS
- Windows 10
- Kali Linux
- VirtualBox
- TCP/IP Networking
